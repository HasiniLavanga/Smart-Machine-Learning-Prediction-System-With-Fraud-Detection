Smart Machine Learning Prediction System with Fraud Detection

📄 Description

Developed a web-based machine learning application using Streamlit to detect fraudulent credit card transactions. The system allows users to upload datasets or use a default dataset, automatically preprocesses the data, and trains a predictive model.

It leverages a Random Forest classifier to identify suspicious transactions and provides real-time insights through an interactive dashboard.

🚀 Key Features 📂 Upload credit card transaction dataset (CSV) 🎯 Automatic detection of fraud column (Class) ⚡ Efficient handling of large datasets using sampling 🤖 Machine learning model training and prediction 📊 Visualizations: Fraud vs Normal distribution Prediction comparison charts Confusion Matrix 📄 Classification report for model evaluation 🔥 Feature importance analysis 📥 Downloadable prediction results 🛠️ Technologies Used Python Pandas, NumPy Scikit-learn Matplotlib Streamlit 🎯 Objective

To build an intelligent system that detects fraudulent credit card transactions accurately and provides easy-to-understand insights through an interactive interface.
